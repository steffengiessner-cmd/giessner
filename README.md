# Giessner Academic Website

A simple personal academic website for GitHub Pages.

## Pages

- Home: `index.html`
- About: `about.html`
- Research: `research.html`
- Teaching: `teaching.html`
- Publications: `publications.html`
- Speaker: `speaker.html`
- Contact: `contact.html`

## Structure

- `styles.css` contains the shared responsive design.
- `app.js` contains the mobile navigation toggle and active page state.
- The site uses plain HTML, CSS, and JavaScript only.

## Edit

Open the HTML files and replace the placeholder text, contact details, publication entries, and profile information with your own content.

## Publish With GitHub Pages

In GitHub:

1. Open the repository settings.
2. Go to **Pages**.
3. Choose **Deploy from a branch**.
4. Select the `main` branch and root folder.
5. Save.
